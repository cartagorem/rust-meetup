# vim + emacs

---

## vim ~/.emacs

Changes won't be evaluated

---

## vim-emacs plugin

 * Select a range
 * Type command `:Emacs`
 * Will evaluate range in Emacs

---

## How?

 * Emacs with server
 * Lines sent with `emacsclient --eval`
 * Elisp expressions evaluated
 * Output is returned
